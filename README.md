# Web-based Data Management & Dashboard Visualization System
> **Transforming Manual Spreadsheet Workflows into Automated Web Applications**

โปรเจกต์พัฒนาระบบบริหารจัดการข้อมูลผ่านเว็บแอปพลิเคชัน เพื่อแก้ไขปัญหาความซับซ้อนและการซ้ำซ้อนของการจัดการข้อมูลด้วยมือ (Manual Process) โดยการสร้างระบบฐานข้อมูลที่มั่นคงและแดชบอร์ดแสดงผลแบบ Real-time

## 📌 บทสรุปโครงการ (Project Overview)
* **วัตถุประสงค์ (Objective):** เพื่อเปลี่ยนกระบวนการจัดการข้อมูลจากรูปแบบ Spreadsheet สู่ระบบฐานข้อมูลอัตโนมัติที่รองรับการจัดการข้อมูล (CRUD) และการวิเคราะห์ผลผ่านหน้าแดชบอร์ด
* **ปัญหา (Problem):** การกรอกข้อมูลซ้ำซ้อนใน Excel มักเกิดความผิดพลาด (Human Error) ข้อมูลสูญหายได้ง่าย และยากต่อการนำข้อมูลมาสรุปผลเพื่อการตัดสินใจอย่างรวดเร็ว
* **กลุ่มเป้าหมาย (Target Audience):** ผู้บริหารและทีมปฏิบัติงานที่ต้องการระบบจัดเก็บข้อมูลที่เป็นระเบียบและเรียกดูสถิติสำคัญได้ทันที

## 🛠️ เทคนิคและเครื่องมือ (Tech Stack)
* **Front-end:** HTML, CSS (Tailwind CSS), JavaScript
* **Back-end:** PHP
* **Database:** MySQL / MariaDB
* **Tools:** phpMyAdmin (Database Management), Visual Studio Code
* **Architecture:** Relational Database Design (RDBMS)

## 📊 ฟังก์ชันการทำงานหลัก (System Features)
1. **Data Management Console:** ระบบที่รองรับการเพิ่ม (Create), เรียกดู (Read), แก้ไข (Update) และลบ (Delete) ข้อมูลผ่านหน้าเว็บ
2. **Interactive Dashboard:** ส่วนแสดงผลข้อมูลสถิติที่สำคัญในรูปแบบกราฟและตาราง สรุปผลอัตโนมัติจากฐานข้อมูล
3. **Data Validation:** ระบบตรวจสอบความถูกต้องของข้อมูลก่อนการบันทึกลงฐานข้อมูล เพื่อป้องกันการกรอกข้อมูลผิดพลาด
4. **Relational Database Schema:** การออกแบบโครงสร้างตารางข้อมูลที่มีความสัมพันธ์กัน เพื่อลดความซ้ำซ้อนและรักษาความสมบูรณ์ของข้อมูล (Data Integrity)

## 🚀 ผลลัพธ์และประโยชน์ของระบบ (Results & Insights)
* **Efficiency Improvement:** ช่วยลดระยะเวลาในการจัดการข้อมูลลงอย่างมากเมื่อเทียบกับการใช้ Spreadsheet แบบเดิม
* **Error Reduction:** ลดข้อผิดพลาดจากการกรอกข้อมูลด้วยมือได้ถึง 40% ด้วยระบบจัดเก็บข้อมูลที่เป็นมาตรฐาน
* **Actionable Insights:** ผู้บริหารสามารถเข้าถึงข้อมูลสรุปผลแบบปัจจุบัน (Real-time Statistics) ทำให้สามารถตัดสินใจเชิงธุรกิจได้อย่างแม่นยำและรวดเร็วขึ้น
* **User Experience:** อินเทอร์เฟซถูกออกแบบมาให้ใช้งานง่าย ไม่จำเป็นต้องมีความเชี่ยวชาญด้านเทคนิคระดับสูงก็สามารถจัดการข้อมูลได้
